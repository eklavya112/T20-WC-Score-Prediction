Project Details:

1. This project contains Score Prediction for T20Is in Men's cricket based on past data which is collected by "cricsheet" and the dataset link is given 
at the beginning.
2. Following steps should be followed to understand it completely :
    (1) Open the World Cup T20 .ipynb file becasue most of the things is done there
    (2) run the application code file .py and it will open up as site using streamlit
    (3) To deploy it on heroku create your heroku account then follow the commands and run it.
